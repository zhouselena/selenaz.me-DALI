AstroRun

AstroRun is a 2D platformer game I wrote with PyGame. It features a little astronaut
creature trying to avoid asteroids and other space-related obstacles. The purpose
of this project was to get familiar with 2D platformers and game dev.
Press on the screenshot to get the link to my GitHub and download it to play!
Specs: Python, PyGame

rEVbot

rEVbot is a multipurpose Discord bot I wrote for UofTHacksVIII with my team. It
features an interactive electric vehicle mini-game that gives users facts about
electric vehicles, a user coin leaderboard, and a function that detects and
filters inappropriate messages.
Press on the screenshot to get the link to the DevPost and add it to your own
Discord server!
Specs: Python, Discord API, Firestore DB, Heroku Cloud
